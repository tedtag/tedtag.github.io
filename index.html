Me and My Skills

Web and UX Design + Frontend Development:

I'm all about style and efficiency. My tools of trade range from Photoshop to Illustrator; from Axure to Balsamiq. I use whatever works and love to research new technologies and how to become a more efficient designer and developer. Lately, I've been enjoying learning about web security including learning how to perofrm and mitigate major web vulnerabilities such as XSS, CSRF, and SQLi attacks.

In the past, I've worked on several class projects using Microsoft Visual Studios and C# with the ASP.NET MVC framework. Server-sided coding with PHP has also become second-nature to me after designing websites utilizing the Facebook Graph API for student organizations at the University of Washington. More recently, I've been focused in solidifying my JavaScript proficiency by learning new frameworks and libraries on top of the ones that I've previously used such as jquery, three.js, Prototype, bootstrap, Popcorn.js, Less.js, and more. I am also currently working on learning AngularJS and Python in my spare time.

Database Migration and Warehousing:

I'm a database guy. Relational databases come naturally to me, and I enjoy looking at database structures and figuring out how to make the as efficient as possible. I enjoy every part of the database design process: from analyzing business specifications to ERD designing with Visio to normalizing the data to coding the queries; from the conceptual phase to the logical phase to the physical phase. Microsoft SQL Server Management Studio is my best friend, and with it I typically feast on Stored Procedures and top it off with our favorite dessert, Database Normalization, with a dash of SQL Server Agent. I've also become familiar with the ETL and Data Warehousing process during a case study and have solid experience working with Microsoft SSIS.

Coding and Programming:

Problem solving is a hobby of mine, and alongside it comes my passion for coding. I enjoy coding and analyzing source code to help out clients in need. During my study abroad in Japan, I learned the Processing programming language in two months time, and created a vibrant visualization of raw doppler radar data to show weather problems in the Osaka Suita city region.

I have taken various programming courses taught in Java and have a solid understanding of data structures and algorithms. If you have time, checkout my GitHub for examples of some of my works!
